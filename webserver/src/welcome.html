Welcome to the Helix Server!

Note: this overview is old. Quite some things have changed. API now under /api/*

This server has access to the Helix database. Overview:

  • / (this page)

  • /overview
    overview of current tasks (based on due_date and deadline)
  • /gettasks?parameter=value
    a GET API-Endpoint to get the current tasks from the database. Parameters:
    • general
    • title
    • description
    • createdAt
    • dueDate
    • deadline
    • id
    • precise(bool)
  • /helloworld
    test side for the server cuz iz kool

API

Besides the GET-Endpoint above (which you can access simply in the address bar of the browser) there are the following POST request endpoints (all parameters like title/description are encapsulated in the request's body):

  • /newtask
    accepts title, description, dueDate, deadline
  • /deletetask
    uses the id attribute
  • /deletealltasks
